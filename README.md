React + firebase

#　機能

### firebase を用いた認証（ログイン、サインイン、ログアウト）

| username       | password |
| -------------- | -------- |
| john@gmail.com | 123123   |
| jane@gmail.com | 456456   |
| jack@gmail.com | 789789   |

## メッセージ送信・画像送信

- リアルタイムに更新

### ユーザー検索

- user id（avatar をクリックすると表示される） を用いてユーザーを検索
- 検索結果をクリックすることで友達追加

### リプライ

- メッセージを右クリックでリプライが可能
- リプライ元までスクロール
